避免多条件并列

Jul 6, 2020 · 1 min · Archai

开发中有时会遇到多个条件，执行相同的语句，也就是多个||这种：

if (status === 'process' || status === 'wait' || status === 'fail') {
  doSomething()
}

这种写法语义性、可读性都不太好。可以通过switch case或includes这种进行改造。

switch case
switch(status) {
  case 'process':
  case 'wait':
  case 'fail':
    doSomething()
}
includes
const enum = ['process', 'wait', 'fail']
if (enum.includes(status)) {
  doSomething()
